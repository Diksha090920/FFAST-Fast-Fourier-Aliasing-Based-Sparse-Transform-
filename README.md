# FFAST-Fast-Fourier-Aliasing-Based-Sparse-Transform-
 Designed and implemented the front-end and back-end architecture of the FFAST( Fast Fourier Aliasing-based Sparse Transform) algorithm in Verilog
Front-end architecture processes an n-length input signal, subsampling it into ’d’ stages, from which the large
n-length DFT coefficients are synthesized using a peeling decoder.
